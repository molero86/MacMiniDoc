# MacMiniDoc — Documentación del Mac Mini M4

Repositorio de documentación de todos los proyectos desplegados en el Mac Mini M4.

## Ver la web de documentación

La documentación se puede navegar como web en **GitHub Pages**:  
👉 `https://tu-usuario.github.io/MacMiniDoc/`

## Desarrollo local

```bash
# Instalar dependencias
pip install mkdocs mkdocs-material

# Servidor de desarrollo (hot-reload)
mkdocs serve

# Construir sitio estático
mkdocs build
```

## Estructura

```
docs/
├── index.md                        # Página principal
├── servidor/                       # Documentación del hardware
└── empresas/                       # Una carpeta por empresa
    ├── noticias-claras/
    └── casa-inteligente/
```
