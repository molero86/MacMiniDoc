# Proyectos — Forja Digital

| Proyecto | Descripción | Estado |
|---|---|---|
| `forja-discovery` | Motor para detectar y puntuar oportunidades de producto | 🚧 En construcción |
| `forja-delivery` | Framework operativo para ejecutar proyectos software de extremo a extremo | 🚧 En construcción |
| `forja-growth` | Stack de lanzamiento, marketing y optimización de adquisición | 🚧 En construcción |
| `forja-ops` | Sistema de seguimiento de portfolio, métricas y decisiones de inversión de tiempo/capital | 🚧 En construcción |

## Orden recomendado de construcción

1. `forja-delivery` (versión Lite para operar el primer proyecto)
2. `forja-ops` (seguimiento y métricas)
3. `forja-discovery` (ampliar generación de oportunidades)
4. `forja-growth` (optimizar adquisición y escalado)

## Proyecto piloto de validación

El primer proyecto ejecutado por Forja Digital será **El Alambique**.  
Si El Alambique llega a producción con este framework, Forja quedará validada como factoría reutilizable.
