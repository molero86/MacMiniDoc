## Estado de detección de RSS

> Durante el desarrollo del Corresponsal se verificará si cada fuente expone un feed RSS/Atom.  
> Si no dispone de RSS, se usará scraping web con `requests` + `BeautifulSoup`.

| Estado | Significado |
|---|---|
| ✅ RSS disponible | Feed RSS/Atom detectado y funcionando |
| 🔄 Scraping web | Sin RSS, se usa scraping |
| 🔍 Por confirmar | Pendiente de verificar |
| ❌ No accesible | Fuente bloqueada o sin contenido estructurado |
